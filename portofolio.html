Skill

Web Development

Web Developer adalah perancang sebuah website / www. Web Developer merancang sebuah website dari awal sekali, menciptakan coding dsb, atau bisa juga seorang Web Developer mengembangkan sebuah website yang sudha ada. Web Developer bisa bekerja secara terikat dengan perusahaan, atau menjadi pekerja lepas (freelancer). Tugas Web Developer juga tidak terbatas di website, bisa juga mengembangkan software, aplikasi, dll.

Web Design

Web designer atau perancang web adalah profesi yang dijalankan oleh orang-orang yang pekerjaannya membuat desain sebuah website. Desain tersebut nantinya bisa dibuka dan dinikmati pada sebuah layar monitor baik melalui komputer, handphone, ataupun tablet. Seorang web designer mulai bekerja membuat desain website begitu dapat permintaan dari perusahaan atau institusi lain.

Mobile Development

developer adalah profesi yang bertugas mengembangkan aplikasi mobile. Secara umum, ada dua sistem operasi berbasis mobile, yaitu Android dan iOS. Dan kemungkinan besar kamu sedang menggunakan perangkat dengan salah satu OS tersebut.
